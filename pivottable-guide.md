## How to Create a PivotTable
1. Click anywhere in your data
2. Insert → PivotTable
3. Drag fields to the four areas:
   - **Rows** — categories you want to group by
   - **Columns** — sub-groups across the top
   - **Values** — numbers to summarize
   - **Filters** — top-level filter for the whole table
4. Right-click a value → Summarize by → SUM / COUNT / AVERAGE
5. Right-click → Show Values As → % of Total / % of Column for comparisons

---

## Golden Rule
> Clean data first. No merged cells, no blank rows, consistent column headers.
> If your source data is messy, your PivotTable will be too.

---

## Common Uses
- Summarize sales by region or category
- Count records by status or type
- Calculate averages across groups
- Compare performance across time periods
